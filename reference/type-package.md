# type: Runtime Type Validation for Functions and Objects

Provides a lightweight type system for 'R'. Types are represented as
composable sets of predicates which a valid object must satisfy.
Variables may be declared with a type, which is checked everytime the
variable is assigned to. Additionally, function arguments and return
values may be annotated with a type, which is validated each time the
function is called.
